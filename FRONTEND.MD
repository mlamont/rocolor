# Notes on Developing the Webpage Frontend

Principles & approach & pseudocode live here.

Like a scratchpad as I build out this webpage.

## Principles

- Minimal tech will facilitate maximal decentralization.
  - No fancy JS frameworks.
  - Single page.
  - Just HTML5. Maybe style at the very bottom.
- Make the code human-readable.
  - Gift to future developers.
  - Gift to future self as I build out a Rockopera Artistry ecosystem.
- Host this myself.
  - I'm paying for a hosting service already.
  - Later explore if it can run an IPFS node, so I can host it there.
